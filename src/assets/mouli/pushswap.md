Pushswap
===

Use Minimouli to run automated tests on your pushswap project.

Installation
---

If you haven't already done so, you'll need to install the Minimouli client.

You can install it by using npm with the following command:

```bash
npm i -g minimouli
```

Or, if you prefer, by using yarn:

```bash
yarn global add minimouli
```

Usage
---

Once installed, run the following command in the root directory of the pushswap project:

```bash
minimouli run
```

The Minimouli client will automatically detect the project and setup the right configuration at the first run. Automated tests are launched immediately after.

Difference
---

Large tests are not tested. They count for 15/20% of the final percentage in the official automated tests.

Issues
---

If you encounter any problem by using Minimouli with this project, don't hesitate to open an issue on the [Github page](https://github.com/minimouli/pushswap/issues).
